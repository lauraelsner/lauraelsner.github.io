It'Sugar - Website Redesign

Role

UX Designer

Launch Date

June, 2015

Project Overview

It'Sugar was already using Magento to host their eCommerce site, but it looked and felt very dated. The goal of this project was to create a responsive, modern website with a particular focus on online gifting and bulk candy purchasing.

Process

  1. User Feedback Sessions
  2. Onsite Discovery (3 Days)
  3. Taxonomy Exercises
  4. Sketching & Prototypes
  5. Responsive Wireframes & Documentation

User Feedback Sessions

Prior to the onsite discovery, I conducted a series of user feedback sessions on the existing itsugar.com website using the 3rd party testing tool usertesting.com. This was an opportunity to do an evaluation of the current site, as well as become accustomed to user shopping behavior for this market. Below you can view the script as well as a small sample of recommendations that I presented to the It'Sugar team during the onsite discovery sessions.

Competitors & Brand Strategy

Vince challenged us to solve for the best approach to present their "dual-gender, global, lifestyle brand". In order to evaluate how the competition was approaching Men vs. Women as navigation and/or separate microsites I created a competitors board using InvisionApp in order to identify structural themes from the sites they were either inspired by or called competition.
